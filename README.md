* Linux

* MacOS

# Instalação

Para instalar este pacote atravês do npm, precisamos rodar o seguinte comando:
 ```bash
 npm i banido -g
 ```

 Caso você queira roda-lo, vai precisar rodar o seguinte comando:
 ```bash
 npx banido
 ```
 ou
 ```bash
 banido
 ```
